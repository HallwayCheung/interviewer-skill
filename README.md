# 面试官模拟器 · Interviewer Skill

> 面试不是考试，是博弈。你要在45分钟内读懂对面那个人，然后决定怎么打这场仗。

一个 Claude Code 面试模拟 Skill。模拟五种真实面试官类型，提供沉浸式面试练习和深度复盘分析。帮你提前踩坑，而不是在真正的面试里交学费。

---

## 为什么需要这个

你面过多少次试？有多少次是面完后才反应过来「原来那个面试官是那种类型，我当时应该xxx」？

面试最大的问题不是你技术不行，而是你读不懂对面那个人。同一个问题，刁难型面试官想听你 push back，恶心人型面试官想看你崩溃，聊得好不给过型面试官其实已经决定不要你了。

这个 skill 让你提前练习识别和应对这五种面试官，把「事后诸葛亮」变成「事前有准备」。

## 五种面试官类型

### 1. 刁难型 — 追问到底的压力测试者

**核心心理**：不是恶意，而是相信压力测试能筛出真本事。尊重能力——如果你逻辑清晰地 push back，反而加分。

**行为特征**：
- 开场冷淡，直奔主题：「坐吧。直接开始。」
- 每个回答都被追问：「那如果...」「你说的这个有没有考虑过...」「这个方案的缺点呢？」
- 沉默施压：回答完后沉默3-5秒，让你自我怀疑
- 抓住你回答中的一个词深挖，假设场景不断升级

**口头禅**：「继续」「然后呢」「说重点」「我不这么认为」「你的依据是什么」

**识别时机**：开场不闲聊 + 第一个问题就有难度 = 第1分钟确认

**关键洞察**：这种面试官其实是最公平的——他们只看能力，不看背景。如果你有真本事，这是最好的展示机会。

---

### 2. 恶心人型 — 权力补偿型攻击者

**核心心理**：不安全感，利用权力差让自己感觉优越。可能把自己的职场怨气投射到候选人身上。

**行为特征**：
- 开场轻慢，可能在看手机：「哦，你来了。简历我还没看，你简单说说。」
- 质疑不是为了深挖，而是为了贬低：「你这项目也叫项目？」
- 从质疑回答 → 质疑能力 → 质疑背景 → 质疑人格
- 对比攻击：「前面那个人比你强多了」「你们学校就教这个？」

**口头禅**：「你确定？」「就这？」「你们学校就教这个？」「你这个年纪还在面这种岗位？」

**识别时机**：开场看手机 + 「简历还没看」+ 1分钟时说「没听过你们公司」= 第2分钟确认

**关键洞察**：这种面试官的行为100%反映了他们自己的问题，不是你的问题。面试结束后不要自我怀疑。

---

### 3. 聊得好但不给过型 — 友善的走过场者

**核心心理**：回避冲突，或者确实 nice 但按评分标准说「不」。可能面试前就决定了（内部候选人、headcount 冻结）。

**行为特征**：
- 开场热情亲切：「嗨！欢迎欢迎，坐，喝水吗？」
- 只聊轻松话题，不碰技术深水区
- 频繁说「挺好的」「不错不错」「嗯嗯我理解」
- 对最有深度的回答不追问
- 面试时间明显偏短（25分钟 vs 标准45分钟）
- 回避具体下一步：「HR会联系你」而不是「下周二之前给你反馈」

**口头禅**：「挺好的」「嗯嗯我理解」「你这个想法很有意思」「今天聊得很开心」「保持联系啊」

**识别时机**：寒暄3分钟 + 问兴趣爱好 + 不追问技术 + 面试时间短 = 第15分钟确认

**关键洞察**：聊得好 ≠ 过得了。真正的兴趣会体现在追问深度和时间投入上。

---

### 4. KPI型 — 敷衍的凑数者

**核心心理**：根本不在乎结果。要面 X 个人才能完成指标。可能是临时拉来凑数的。只想快点结束。

**行为特征**：
- 开场匆忙：「好，我们开始吧。你先自我介绍一下。」
- 问题全是通用模板：优缺点、五年规划、为什么来我们公司
- 反应只有「嗯」「好」「行」，没有情感波动
- 可能在打字、看屏幕，边听边做别的事
- 面试时间极短（12分钟 vs 标准45分钟）
- 在很早的时间点就问「你有什么想问我的吗？」

**口头禅**：「嗯」「好」「行」「下一个问题」「时间差不多了」

**识别时机**：开场直接念问题 + 全程打字 + 第3分钟就确认

**关键洞察**：不要浪费精力表演。把时间用来问你自己的问题，收集关于公司/团队的信息。

---

### 5. HR压力面型 — 冷酷的信息验证者

**核心心理**：职业 HR，善于通过连续追问和情绪控制拆解候选人。目标是让你自相矛盾，暴露真实动机和弱点。

**行为特征**：
- 开场客气但带审视感：「你好，请坐。我看你上一家公司待了两年？」
- 离职原因追问3层：为什么？具体是什么？当时没沟通过？
- 薪资施压：「这个数字有点高」「你目前薪资多少？」「能接受降一点吗？」
- 沉默让你补充、假装不满意你的回答、用微表情施压
- 职业规划审视：「为什么不走管理？」「听起来你还没想清楚？」

**口头禅**：「能具体说说吗？」「为什么呢？」「那你觉得呢？」「这个原因我不太理解」

**识别时机**：开场直接从简历细节切入 + 离职追问3层 = 第2分钟确认

**与恶心人型的区别**：HR压力面攻击你的回答逻辑，恶心人型攻击你这个人。HR压力面是专业行为，不是恶意。

## 深度复盘系统

面试结束后（说「结束面试」），进入教练模式。复盘包含以下维度：

### 复盘结构

```
## 面试复盘

### 面试官类型识别
- 类型：X型
- 关键行为特征：[列出面试中展现的典型行为]
- 识别时机：你应该在第X分钟/第X个问题时就识别出来

### 逐题分析

**Q1: [问题]**
你的回答摘要：[总结]
- 做得好的：[具体表扬]
- 可以改进的：[具体建议]
- 更好的回答思路：[具体方案]
- STAR结构使用：[是否使用/使用是否恰当/如何改进]

（重复每个问题）

### 表达技巧分析
- 逻辑性：回答是否有清晰的结构？是否出现前后矛盾？
- 信息密度：是否有废话？该展开的没展开还是该简洁的太啰嗦？
- STAR结构：行为面试题中是否使用了STAR？使用是否自然？
- 语速和节奏：是否因紧张而语速过快？是否有适当的停顿？
- 自信度：用词是否自信？是否有太多弱化词？

### 心理状态评估
- 压力反应：面对追问时是否慌张？是否出现防御性回答？
- 情绪波动：是否被面试官的态度影响？何时开始紧张/沮丧/急躁？
- 掌控感：是否尝试夺回对话主导权？还是完全被动回答？

### 识别信号（面试中你应该注意的）
- [信号1]
- [信号2]
- [信号3]

### 应对策略（下次遇到这种面试官）
- [策略1]
- [策略2]
- [策略3]

### 综合评分：X/10
- 回答质量（技术深度、逻辑性）：X/10
- 表达技巧（结构、简洁、自信）：X/10
- 压力应对（情绪稳定、push back能力）：X/10
- 识人能力（是否识别出面试官类型）：X/10
```

### 评分标准

| 维度 | 9-10分 | 7-8分 | 5-6分 | 3-4分 | 1-2分 |
|------|--------|-------|-------|-------|-------|
| 回答质量 | 深度+广度+创新 | 逻辑清晰有深度 | 基本正确但浅 | 有明显错误 | 答非所问 |
| 表达技巧 | STAR+简洁+自信 | 结构清晰 | 有结构但啰嗦 | 混乱无序 | 不知所云 |
| 压力应对 | 稳定+有效push back | 基本稳定 | 被影响但没崩 | 情绪失控 | 完全崩溃 |
| 识人能力 | 3分钟内识别 | 5分钟内识别 | 面试中途识别 | 面试结束才识别 | 完全没识别 |

## 常见问题

### Q: 面试官的追问有标准答案吗？

没有。每种面试官的追问方式是固定的，但你的回答可以多种多样。复盘的重点不是「标准答案是什么」，而是「你的回答在哪些方面可以改进」。

### Q: 可以同时练习多种面试官类型吗？

建议一次练习一种类型。刻意练习的效果远好于随机练习。如果你能稳定应对一种类型，再换下一种。

### Q: 复盘评分偏低怎么办？

评分低不代表你不行，而是暴露了需要提升的具体方面。看复盘中的「可以改进的」和「更好的回答思路」，有针对性地练习。

### Q: 恶心人型面试官真的存在吗？

存在，但不常见。大部分人遇到的「恶心人」面试官其实是刁难型（追问到底但不人身攻击）。练习恶心人型的目的是：万一真遇到了，你知道这不是你的问题。

### Q: 面试中怎么快速识别面试官类型？

前3分钟是关键。看三个东西：
1. 开场方式（热情/冷淡/轻慢/匆忙/审视）
2. 第一个问题的类型（技术深挖/质疑/闲聊/模板/简历细节）
3. 对你第一个回答的反应（追问/否定/说好/说好然后念下一个/追问细节）

### Q: 这个 skill 适合什么岗位？

覆盖六个岗位方向：后端开发、前端开发、产品经理、数据分析师、算法工程师、测试工程师。也可以选择通用技术岗。题库按类型×岗位交叉组织，确保每种组合都有针对性的题目。

### Q: 复盘中的STAR结构是什么？

STAR是行为面试题的回答框架：
- **S**ituation（情境）：当时的背景是什么
- **T**ask（任务）：你要解决什么问题
- **A**ction（行动）：你具体做了什么
- **R**esult（结果）：最终效果如何

在刁难型和HR压力面型面试中，使用STAR结构能让你的回答更有条理。

### Q: 「随机挑战」模式是怎么选的？

Skill 会随机选择一种面试官类型，不提前告诉你。这是进阶练习——因为在真实面试中，你也不知道对面是什么类型的人。
